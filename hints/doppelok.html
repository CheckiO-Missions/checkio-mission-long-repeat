You have a few things left. Wrap this in a function expression and return the maximum value among all result values, that will be the answer. It is also desirable to protect our code from an empty string ;-). Therefore, we will use the "default" argument in the max() function, if the function cannot calculate the maximum value, it will return "0". 
def long_repeat(line: str) -> int:
    result = []
    lenline = len(line)
    while line:
        line = line.lstrip(line[0])
        result.append(lenline - len(line))
        lenline = len(line)
    return max(result, default=0)  # we return the maximum value
