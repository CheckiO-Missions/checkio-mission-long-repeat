您还剩下几件事。用函数表达式表示，返回所有结果值中的最大值。 值中的最大值，这就是答案。我们还希望保护我们的代码不出现空字符串;-)。 因此，我们将在max()函数中使用 "default "参数。 将返回 "0"。 
def long_repeat(line: str) -> int:
    result = []
    lenline = len(line)
    while line:
        line = line.lstrip(line[0])
        result.append(lenline - len(line))
        lenline = len(line)
    return max(result, default=0)  # we return the maximum value
