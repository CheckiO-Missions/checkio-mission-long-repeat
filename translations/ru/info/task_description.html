Существует четыре миссии связанные с анализом частей строки. Все они были созданы за один день и не требуют более одного дня для своего решения. Эти миссии можно с легкостью преодолеть посредством простого перебора символов. Но, возможно, есть вариант получше? (У Вас может еще не быть доступа ко всем этим миссиям, но по мере открытия островов на карте он будет предоставлен)

Это первая миссия из серии. Вам необходимо найти длину самой длинной подстроки, которая состоит из одинаковых букв. Например, строка "aaabbcaaaa" состоит из четырех подстрок с одинаковыми буквами "aaa", "bb","c" и "aaaa". Последняя подстрока является самой длинной, что и делает ее ответом.

Входные данные: Строка.

Выходные данные: Целое число.
